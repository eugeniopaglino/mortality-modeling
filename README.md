# Mortality Modeling Workshop

The notes are in the R folder. See mortality_modeling.Rmd for the RMarkdown file and mortality_modeling.html for the knitted version. [Here](https://htmlpreview.github.io/?https://github.com/eugeniopaglino/mortality-modeling/blob/main/R/mortality_modeling.html) is a rendered version you can open directly in your browser. If you want to experiment with the R code, I would recommend creating an R script rather than working directly with the Rmd file. For the last part of the notes (which are entirely optional), you will need to install the `cmdstanr` package.
